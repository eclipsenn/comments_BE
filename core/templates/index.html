API:

1. /{username}/create_comment - create a new comment for the given entity id;

2. /{username}/change_comment - change comment for the given comment id;

3. /{username}/delete_comment - delete comment for the given comment id;

4. /{username}/lvl1?entity_id=3 - get all level 1 comments for entity with id=3;

5. /{username}/get_children?entity_id=4 - get all children for the given entity.

6. /{username}/get_full_tree?entity_id=4 - get full tree(including root) for the given root id.

7. /{username}/get_comments - get all the comments for the given user;

8. /{username}/get_history - get history of comments for the given user

9. /{user}/search_history - show search history with re-download option
